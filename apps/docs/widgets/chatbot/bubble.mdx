To install an Agent on your website using the Bubble widget use the follwing code snippet:

```html
<script type="module">
  import Chatbox from 'https://cdn.jsdelivr.net/npm/@chatsappai/embeds@1.0.231/dist/chatbox/index.min.js';

  Chatbox.initBubble({
    agentId: 'YOUR_AGENT_ID',
  });
</script>
```

Replace `YOUR_AGENT_ID` with the ID of your Agent

</Note>
